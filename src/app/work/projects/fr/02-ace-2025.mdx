## Présentation

ACE 2025 est une plateforme web conçue pour l'Antilles Cyber Escape, un événement d'escape game cybersécurité organisé pour l'Université des Antilles.

Le projet regroupe un site d'inscription et une plateforme CTFd personnalisée. Le site permet aux participants de créer leur compte, de vérifier leur adresse email, de créer ou rejoindre une équipe, de choisir un créneau et de suivre les informations liées à l'événement. Une interface administrateur permet aussi de gérer les équipes, les salles, les exports CSV et les notifications.

La partie CTFd synchronise les équipes, les scores et les informations de jeu avec le site d'inscription. Elle intègre plusieurs épreuves de cybersécurité et des plugins personnalisés pour adapter l'expérience au format de l'événement.

## Technologies utilisées

- **Next.js**: application front-end pour les pages d'inscription, d'équipe et d'administration.
- **Node.js / Express**: API back-end pour l'authentification, les équipes, les créneaux et l'administration.
- **PostgreSQL**: base de données principale du site d'inscription.
- **Redis**: cache et support technique pour certains traitements.
- **CTFd**: plateforme de challenge cybersécurité personnalisée.
- **Docker**: orchestration des services front-end, back-end, base de données et plateforme CTF.
